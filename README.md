# Frontend Starter kit

Установка зависимостей

```sh
npm i
```

Запуск локального сервера [http://localhost:8080](http://localhost:8080)

```sh
npm start
```

Запуск локального сервера на другом порту

```sh
PORT=1234 npm start
```

Создание минифицированного билда

```sh
npm run build
```
